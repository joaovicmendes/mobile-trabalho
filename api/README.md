# API de Cursos

## Courses
Payload para adicionar um novo curso:

```json
{
    "title": "Título",
    "instructorId": "eO4pC6odTu00TT2AczJJ",
    "description": "Descrição",
    "rating": 4,
    "categoryId": "KWc45Hp0VgGaTqdD1Fb4",
    "schedule": "none",
    "videoUrl": "none",
    "thumbnail": "none"
}
```

## Users
Payload para adicionar um novo usuário:

```json
{
    "name": "Nome do Usuário",
    "username": "user_name",
    "email": "email@email.com",
}
```

## Category
Payload para adicionar uma nova categoria:

```json
{
    "name": "Nome da Categoria",
    "hexColor": "#FFFFFF"
}
```
